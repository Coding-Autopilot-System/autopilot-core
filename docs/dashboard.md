# Autopilot Dashboard

This dashboard provides a consolidated view of automation status and operational health.

## Live status
- `docs/status.md` is updated daily by the docs workflow.
- Use it as the source of truth for queued and blocked issues.

## Signals
- Intake health: check recent workflow runs for `autopilot-create-issue.yml`
- Operator health: check recent workflow runs for `autopilot-operator.yml`
- Runner health: check `ci-autopilot` for runner offline issues
