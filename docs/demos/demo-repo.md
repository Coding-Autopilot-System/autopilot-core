# Demo Repo Walkthrough

This demo shows the full Autopilot intake loop using a failing CI workflow.

## Steps
1) Create a demo repo from `templates/demo-repo`.
2) Trigger `Demo CI` workflow.
3) Confirm an `autofix + queued` issue is created by intake.
4) Run the operator to see it pick up the issue.

## Notes
Use this demo in sales and onboarding to show the end-to-end flow.
